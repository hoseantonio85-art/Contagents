# Norm — AI Risk Management System

Norm is an AI-powered system for operational risk management.

It helps users:
- understand current risk situation
- identify why risks are changing
- decide what actions to take

This is not a dashboard.
This is a decision-making interface.

The system combines:
- structured data (risks, incidents, measures)
- AI analysis and explanations
- interactive UI

Key principle:
The interface should guide the user, not just display information.

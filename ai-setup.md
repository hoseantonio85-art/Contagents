# AI Setup Instructions

When generating UI:

1. Always start from user goal:
   - What is happening?
   - Why?
   - What should user do?

2. Avoid dashboard-style layouts.

3. Prefer:
   - clear hierarchy
   - minimal components
   - meaningful grouping

4. Always include:
   - context (AI summary)
   - main visualization
   - actionable content

5. Lists should:
   - be filtered
   - show top items first
   - be actionable

6. Tone:
   - simple
   - friendly
   - non-bureaucratic

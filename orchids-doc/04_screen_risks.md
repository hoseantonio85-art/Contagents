# Screen: Risks List


Goal:

Show current risk situation and allow quick actions.


Structure:


1. Header

- title: "All risks"

- tabs: Active / Analysis / Archive


2. AI summary block

- short explanation of current situation


3. Risk signals (top cards)

- New risks

- High risks

- Re-evaluated


4. Risk list

- card-based

- each card includes:

  - level

  - name

  - potential loss

  - actual loss

  - strategy

  - related object


5. Floating primary action

- "Register risk"


Important:

- this is not a static list

- it must feel dynamic and alive

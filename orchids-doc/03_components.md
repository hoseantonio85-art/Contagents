# UI Components


## Cards

- soft shadows

- rounded corners (12–16px)

- minimal borders


## Tags

- transparent background

- color-coded:

  - green = low

  - yellow = medium

  - red = high


## Buttons

- primary = green

- secondary = light / outline

- avoid multiple strong buttons


## Lists

- not tables

- card-based

- each item actionable


## AI blocks

- slightly tinted background

- friendly tone

- always suggest next step
